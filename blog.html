localSorage ও sectionStorage এর মদ্ধে পার্থক্য কী?

Answer:

localStorage ও sectionStorage এই দুই টাই হল Browser এর API
sectionStorage - প্রায় সকল browser এ কোনো ডাটা store করে রাখতে Browser এ Application Tab এ sectionStorage and localStorage নামে একটা option থাকে। এবং সেখানে কিছু data Store করে রাখা যায় । sectionStorage এ কোনো Data রাখলে সেই সকল data ঠিক ততক্ষন থাকবে যতক্ষন অই Browser এর অই Tab টি Open থাকবে, অর্থাত আমরা বলতে পারি sectionStorage হল Temporary data রাখতে পারে ।

localStorage - sectionStorage এর মতই localStorage কাজ করে, কিন্তু localStroge এ কোনো Data রাখলে সেই Data Temporary চলে যায় না, localStorage এ Data রাখলে সেই সকল data localStorage এ Store হয়ে থাকে । এবং পরবর্তীতে সেই সকল data Use করা যায় ।

Gobal Scope এর সাথে Blog Scope এর মদ্ধে Difference কি?

Answer:

Gobal Scope: Gobal Scope এর মদ্ধে কোনো Code রাখলে সেই Code কে Gobal অর্থাত JS File এর সব জায়গা থেকে Access করা যায় ।

Blog Socpe: Blog Scope holo Curly brackets {} এর ভিতরের কে বুজায়, এই Curly brackets এর ভিতরে কোনো Code রাখলে সেই Code কে শুধু সেই Curly brackets এর ভিতর থেকে Access করা যার । Gobal (JS File) এ সেই সকল Code কে Access করা যায় না ।

JavaScript এর Event Loop কী ভাবে কাজ করে ?

Answer:

javaScripr এর Event Loop JS Code run করতে একটি গুরুত্বপূর্ণ ভুমিকা পারন করে । JavaScript এ function call করলে সকল function একটার পর একটা Serial Wisa Execute হতে থাকে । প্রথমে সেই সকল code JS এর একটি memory তে Save হয় তারপর সেখান থেকে Execute হয়ে Queue (js engin part) এর মদ্ধে পাঠিয়ে দেয় । যেই code Execute হতে যত কম সময় লাগে সেই code তত কম সময়ে Queue এর মদ্ধে চলে আসে । এবং সেখান থেকে Code গুলো Seial Wise Prient হতে থাকে । এই Event Loop এর কারনে JS Engin Code গুলোকে Brack না করেই চালাতে থাকে ।

JavaScript এ কত ভাবে আমারা undifine পেতে পারি ?

Answer:

JavaScript এ আট ভাবে আমরা undifine পেতে পারিঃ

১। কোনো Varibale initialized না হলে undifine হয়।

২। কোনো function Declare করার পর যদি ঐ function থেকে return না করা হয় তাহলে undifine হয় ।

৩। কোনো function Declare করার সময় যদি Parameter থাকে এবং ঐ function কে call করার সময় সঠিক ভাবে parameter pass না করা হয় তাহলে undifine হয় ।

৪। function এ return করার সময় যদি return এর right side a কোনো কিছু না দেওয়া হয় তাহলে undifine হয় ।

৫। কোনো Object Declate করার পর সেই object এর এমন কোনো একটা property কে যদি call করা হয় সেই property ঐ object এর মদ্ধে নেই তাহলে undifine হয়।

৬। কোনো Array Declate করার পর সেই Array এর এমন কোনো একটা element এর index কে যদি access করা হয় সেই element এর index ঐ Array এর মদ্ধে নেই তাহলে undifine হয়।

৭। কোনো Array Declate করার পর সেই Array এর কোনো একটা element কে যদি deleted করা হয় এবং সেই element এর index কে যদি call করা হয় তাহলে undifine হয়।

৮। কোনো একটা Variable Declare করার সময় যদি ঐ Variable এর মান undifine set করে দেওয়া হয় তাহলে undifine হয় ।
